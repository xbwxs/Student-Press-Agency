# 📰 岳阳县二中学生记者社 — 官方网站 & 管理后台

[![Built with Supabase](https://img.shields.io/badge/Built%20with-Supabase-3ECF8E?logo=supabase)](https://supabase.com)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

岳阳县第二中学学生记者社全栈项目，包含对外展示网站与内部管理后台，覆盖文章发布、招新报名、部门协作、财务记录、每日签到等完整社团运营场景。

| 站点 | 链接 |
|------|------|
| 🏠 主站（社团展示） | `index.html` |
| 📰 校园新闻 | `news.html` |
| ✍️ 招新报名 | `recruitment.html` |
| 🔧 管理后台 | `admin.html` |

## 📦 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | HTML5 + CSS3 + 原生 JavaScript（无框架） |
| 后端 | [Supabase]（BaaS） |
| 数据库 | PostgreSQL（Supabase 托管） |
| 认证 | Supabase Auth（邮箱密码登录） |
| 存储 | Supabase Storage（可选，图片上传） |
| 部署 | 静态托管（Cloudflare Pages / Vercel / GitHub Pages） |

##✨ 功能总览

###🏠 主站（前端展示）

- 🎨 现代化 Hero 首屏 + 动态背景动画
- ⏰ 招新倒计时（精确到秒，联动数据库开关）
- 📰 最新校园动态（置顶文章优先、搜索筛选）
- 🏢 六大部门介绍
- 📜 社团规章制度（章程/义务/奖惩）
- 📊 实时统计数据（滚动数字动画）
- 🌙 暗色模式切换
- 📱 响应式设计 + 底部胶囊导航

### 🔧 管理后台

#### 📰 文章管理
- 写文章、编辑、删除、发布/草稿
- 文章置顶、标题搜索、分类筛选
- 实时 HTML 预览
- 权限分级：社长/行政秘书/副社长全部权限，采访部/策划部/后勤部仅草稿

#### 📝 招新报名审核
- 查看、通过、拒绝、删除报名
- 批量通过、批量删除、清除已拒绝
- CSV 导出
- 详情弹窗查看完整信息

#### 👥 成员管理
- 单个添加 / 批量导入成员
- 修改名称、部门、班级、QQ、电话
- 角色权限分配
- 删除成员

#### 💰 财务管理
- 收支记录新增/删除
- 总收入/总支出/余额统计（全员可见）

#### 📬 意见箱
- 分类提交（建议/吐槽/表扬/想法）
- 匿名开关、点赞
- 社长回复、置顶、搜索筛选

#### 👥 社员墙
- 按部门分组展示成员信息
- 在线状态（10 秒心跳，15 分钟阈值）
- 搜索、排序、点击查看详情
- 导出通讯录
- 编辑个人班级/QQ/电话

#### ✅ 每日签到
- 月历视图 + 签到标记
- 连续签到天数统计
- 今日签到排行榜
- 本周签到柱状趋势图

#### 📋 操作日志
- 最近 200 条操作记录

#### ⚙️ 网站设置
- 首页统计数字修改
- 招新报名通道开关（仅超管）

#🗄️ 数据库结构

| 表名 | 用途 |
|------|------|
| `articles` | 文章（含置顶 `pinned`） |
| `recruitment` | 招新报名记录 |
| `finance_records` | 财务收支 |
| `activity_log` | 操作日志 |
| `profiles` | 成员资料（含 `class_name`/`qq`/`phone`/`last_seen`） |
| `settings` | 网站设置（统计数字、报名开关） |
| `suggestions` | 意见箱（含分类/点赞/回复/置顶） |
| `checkins` | 签到记录（含补签） |
| `media_items` | 素材库（图片/视频链接） |
